# 🌐 Telegram Domains Reference

> Полный справочник рабочих, служебных и инфраструктурных доменов экосистемы Telegram. 
> Предназначен для документации, настройки систем фильтрации трафика, прокси и корпоративных файрволов.

Основные и системные домены
* telegram.org — Главный сайт мессенджера. Используется для загрузки клиентов, авторизации приложений и доступа к основной документации API.
* telegram.me — Основной рабочий домен для коротких веб-ссылок на публичные каналы, чаты, ботов и профили пользователей.
* t.me — Исторический короткий домен для ссылок (используется внутри экосистемы приложений).
* tdesktop.com — Официальный сайт и репозиторий для скачивания десктопной версии клиента (Telegram Desktop).
* web.telegram.org — Домен веб-версий мессенджера (клиенты WebK и WebA).
* telegram.dog — Официальный служебный домен-редирект для коротких ссылок.
* telega.one — Официальное зеркало для веб-ссылок, созданное для обхода блокировок.
* telegram.space — Дополнительный служебный домен infrastructure.

Сети доставки контента (CDN) и медиафайлы
* telegram-cdn.org — Главный домен распределенной сети доставки контента (CDN). Отвечает за быструю загрузку картинок, видео, голосовых сообщений и файлов.
* cdn-telegram.org — Резервный домен для распределения медиаданных мессенджера.

Инструменты, авторизация и веб-сервисы
* tx.me — Служебный домен для бесшовной интеграции Mini Apps (веб-приложений внутри Telegram), авторизационных ботов и внутренних редиректов.
* comments.app — Официальный виджет авторизации через Telegram и сервис для организации комментариев на внешних сайтах.
* discuss.app — Домен платформы обсуждений и авторизации.
* telegra.ph — Минималистичный издательский инструмент (Telegraph) для создания и верстки статей со встроенной поддержкой Instant View.
* graph.org — Официальное зеркало и API-платформа для сервиса Telegraph.
* telesco.pe — Автономный веб-сервис для просмотра публичных видеосообщений («круглых видео») из каналов без авторизации в мессенджере.

Разработка и конкурсы
* tg.dev — Портал технической документации для разработчиков, дизайнеров интерфейсов и создателей стикеров.
* contest.com — Официальная площадка для проведения регулярных конкурсов Telegram (для разработчиков, QA и дизайнеров).

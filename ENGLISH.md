# 🌐 Telegram Domains Reference

> Complete reference of active, utility, and infrastructure domains within the Telegram ecosystem.
> Designed for documentation, traffic filtering systems, proxies, and corporate firewalls.

Main and System Domains
* telegram.org — Main Telegram website. Used for downloading clients, application authorization, and accessing core API documentation.
* telegram.me — Primary active domain for short web links to public channels, chats, bots, and user profiles.
* t.me — Legacy short link domain (currently restricted in standard web browsers but actively used within app ecosystems).
* tdesktop.com — Official website and repository for downloading the desktop client (Telegram Desktop).
* web.telegram.org — Domain for Telegram web clients (WebK and WebA interfaces).
* telegram.dog — Official utility redirection domain for short links.
* telega.one — Official web link mirror created to bypass regional blocks and restrictions.
* telegram.space — Secondary internal infrastructure domain.

Content Delivery Networks (CDN) and Media Files
* telegram-cdn.org — Primary domain for the distributed Content Delivery Network. Responsible for fast loading of images, videos, voice notes, and files.
* cdn-telegram.org — Backup domain for the messenger's media data distribution.

Tools, Authorization, and Web Services
* tx.me — Utility domain for seamless integration of Mini Apps (in-app web applications), auth bots, and internal routing.
* comments.app — Official widget for Telegram-based authorization and comment management on external websites.
* discuss.app — Domain for the discussions and authorization platform.
* telegra.ph — Minimalist publishing tool (Telegraph) for creating articles with native Instant View support.
* graph.org — Official mirror and API platform for the Telegraph service.
* telesco.pe — Standalone web service for viewing public video messages ("round videos") from channels without signing in.

Development and Contests
* tg.dev — Technical documentation portal for developers, interface designers, and sticker creators.
* contest.com — Official platform for regular Telegram competitions targeting developers, QA engineers, and designers.
